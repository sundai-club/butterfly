# Butterfly Browser Test Plans

Use these files for live, no-post browser testing:

- `setup.md`: shared Chrome/CDP setup and deterministic runtime shim
- `linkedin.md`: LinkedIn feed, post page, person feed, company feed
- `producthunt.md`: Product Hunt comment and reply composers
- `twitter-x.md`: X/Twitter reply composers
- `reddit-old.md`: old Reddit top-level and reply textareas

Each platform file records its matrix, per-test checks, error-path checks, and execution results.
